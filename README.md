# Fashion Recommendation System (FRS)

## 📌 Overview
  This project is a Fashion Recommendation System that suggests similar fashion items based on product data. The system uses similarity-based logic to recommend items from a dataset.

## 🚀 Features
  * Recommend similar fashion products
  * Interactive UI using Streamlit
  * Uses dataset files (styles.csv, images.csv)
  * Fast response using stored data (pickle)

## 🛠 Tech Stack
  * Python
  * Pandas
  * Streamlit
  * Pickle

## ⚙️ How It Works
  1. Load dataset from CSV files
  2. Process product data
  3. Use similarity logic to find related items
  4. Display recommendations through Streamlit interface

## ▶️ How to Run
  pip install -r requirements.txt
  streamlit run app.py

## 📊 Dataset
  Dataset contains fashion product details such as styles and images.
